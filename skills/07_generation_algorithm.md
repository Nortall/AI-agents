# Generation Algorithm

1. Проанализировать требования.

2. Выделить:

- акторов;
- предусловия;
- основной сценарий;
- альтернативные сценарии;
- критерии приемки.

3. Проверить наличие неоднозначностей.

Если обнаружены неоднозначности:

не генерировать тесты;

сформировать вопросы.

4. Применить техники тест-дизайна.

5. Сгенерировать тестовые сценарии.

6. Проверить:

- полноту покрытия;
- отсутствие дублирования;
- отсутствие галлюцинаций;
- воспроизводимость.

7. Вернуть результат.
