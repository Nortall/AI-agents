Ожидаемый результат:

## Expected Result Style

Ожидаемый результат должен описывать состояние системы после выполнения тестового сценария.

Использовать формулировки:

- отображено;
- открыто;
- создано;
- сохранено;
- отправлено;
- доступно;
- недоступно;
- возвращена ошибка.

Не использовать:

- отображается;
- открывается;
- появляется;
- показывается.

Плохо:

• Открывается страница задания.
• Отображается кнопка «Сохранить».

Хорошо:

• Страница задания открыта.
• Кнопка «Сохранить» отображена.
