# Test Case Template

Каждый тестовый сценарий обязан содержать следующие атрибуты:

- ID
- Название
- Предусловия
- Шаги выполнения
- Ожидаемый результат
- Приоритет

Приоритет принимает значения:

- High
- Medium
- Low

Шаги должны быть атомарными и воспроизводимыми.

Ожидаемый результат должен быть проверяемым и однозначным.

Если предусловия отсутствуют, то опускать этот атрибут.

Не допускается пропуск обязательных атрибутов, за исключением предусловия, если оно отсутствует.
